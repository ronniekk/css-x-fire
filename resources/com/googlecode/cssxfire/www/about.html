CSS-X-Fire (1.4)

Requirements

This plugin requires Firefox with Firebug.

Get connected

Download and install the corresponding browser plugin - cssxfire.xpi (1.4) - to connect Firebug CSS editor with the IDE plugin.
Firebug Lite users: CSS-X-Fire bookmarklet (drag and drop the link to the bookmarks toolbar).

Using

Use Firebug CSS editor to edit and add style properties.
 
The IDE toolwindow shows the changes made in Firebug in a tree view, categorized by file and CSS selector, provided that the
CSS selectors exist in the project opened. From here you can apply or discard changes on an entire file, selector or on a specific property.
 
The property nodes - the tree leafs - will either be blue (modified), green (new) or gray (invalid). They are also navigatable, e.g.
double-click or press enter key on a valid property to show the corresponding source code.
